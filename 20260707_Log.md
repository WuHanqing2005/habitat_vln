## 操作记录

### 1. 确认数据集来源

访问了 [matterport/habitat-matterport-3dresearch](https://github.com/matterport/habitat-matterport-3dresearch) 仓库，并参考了官网 https://aihabitat.org/datasets/hm3d/ 的说明，确定下载 v0.2 版本中的 minimal 相关数据。

### 2. 下载所需压缩包

按照 README 中的链接，下载了以下 4 个压缩包：

| 文件名 | 类型 | 下载链接 | 大小 |
|---|---|---|---|
| hm3d-minival-glb-v0.2.tar | glb | https://api.matterport.com/resources/habitat/hm3d-minival-glb-v0.2.tar | 464M |
| hm3d-minival-habitat-v0.2.tar | habitat | https://api.matterport.com/resources/habitat/hm3d-minival-habitat-v0.2.tar | 390M |
| hm3d-minival-semantic-annots-v0.2.tar | semantic-annots | https://api.matterport.com/resources/habitat/hm3d-minival-semantic-annots-v0.2.tar | 240.6M |
| hm3d-minival-semantic-configs-v0.2.tar | semantic-configs | https://api.matterport.com/resources/habitat/hm3d-minival-semantic-configs-v0.2.tar | 30K |

下载完成后，本地得到 4 个压缩包。

### 3. 将文件移动到项目目录

在 WSL 中创建目标目录，并把下载好的 HM3D 压缩包移动到项目的数据目录：

```bash
mkdir -p /mnt/f/habitat_vln/data/scene_datasets/hm3d/
mv /mnt/c/Users/wuhanqing/Downloads/hm3d-minival-*.tar /mnt/f/habitat_vln/data/scene_datasets/hm3d/
```

### 4. 在 WSL 中解压数据包

进入目标目录后，依次解压四个 tar 包，保证文件结构符合 Habitat 仿真环境的要求：

```bash
cd /mnt/f/habitat_vln/data/scene_datasets/hm3d/

tar -xvf hm3d-minival-glb-v0.2.tar
tar -xvf hm3d-minival-habitat-v0.2.tar
tar -xvf hm3d-minival-semantic-annots-v0.2.tar
tar -xvf hm3d-minival-semantic-configs-v0.2.tar
```

### 5. 验证解压结果

使用以下命令检查 `.glb` 文件是否正确落在目标目录结构中：

```bash
find . -name "*.glb"
```
